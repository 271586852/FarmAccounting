# 图标显示问题 - 解决方案

## 问题原因
微信小程序对SVG的支持有限，内联SVG可能无法正常显示。

## 解决方案

### 方法一：使用PNG图片（推荐）

请将以下SVG文件转换为PNG格式（64x64px或128x128px），并保存到 `miniprogram/image/` 目录：

1. **记账图标**
   - `record.png` - 灰色版本（未选中）
   - `record-active.png` - 绿色版本（选中，颜色：#1aad19）

2. **统计图标**
   - `statistics.png` - 灰色版本（未选中）
   - `statistics-active.png` - 绿色版本（选中，颜色：#1aad19）

3. **日历图标**
   - `calendar.png` - 绿色版本（颜色：#1aad19）

4. **删除图标**
   - `delete.png` - 红色版本（颜色：#ff4444）

### 转换工具
可以使用以下在线工具将SVG转换为PNG：
- https://svgtopng.com/
- https://convertio.co/zh/svg-png/
- https://cloudconvert.com/svg-to-png

### 转换步骤
1. 打开转换工具
2. 上传对应的SVG文件（在 `miniprogram/image/` 目录中）
3. 设置尺寸：64x64 或 128x128
4. 对于选中状态的图标，需要先修改SVG中的颜色为绿色（#1aad19）
5. 下载PNG文件
6. 将PNG文件保存到 `miniprogram/image/` 目录

### 临时方案
如果暂时无法转换，代码中已使用image标签，可以先使用占位图片或emoji字符。
